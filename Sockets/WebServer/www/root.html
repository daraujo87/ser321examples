You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=...&num2=... -- returns the multiplication of the two given values
  • /github?query=... -- queries the GitHub API with the given query
  • /cypher?string=...&seed=... -- applies a cypher to the given string using an integer as seed
  • /decypher?string=...&seed=... -- decyphers a string using the given seed. Try using the same seed used in /cypher
